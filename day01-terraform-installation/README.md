On the first day of the challenge, I focused on setting up a solid foundation for my Terraform journey. 
### Key tasks included:


```
Installing Terraform (v1.14.7) on my local system
Configuring the AWS CLI with access key, secret key, default region, and output format
Verifying AWS credentials using aws sts get-caller-identity
Ensuring that Terraform and AWS CLI were correctly integrated for seamless deployments
```
---

This setup ensures that all subsequent tasks can leverage a fully functional Terraform environment, ready for provisioning and managing cloud infrastructure.
